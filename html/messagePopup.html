Votre navigateur bloque les popup. Cela vous empêchera d'utiliser des sous-titres dans legolang.
Vous pouvez accepter les pop-up depuis le site legolang.univ-lille3.fr/.

pour Firefox :

Vous devez avoir ce message :

Cliquez sur "Option et validez la première ligne " autoriser les popups pour legolang.univ-lille3.fr". :

Pour Google Chrome :

À droite de votre barre d'adresse vous avez cette icone :

Cliquez dessus et validez l'option " toujours afficher les fenêtres pop-up de legolang.univ-lille3.fr ".

Pour Opera :

À droite de votre barre d'adresse vous avez cette icone :

Cliquez dessus et validez l'option " toujours autoriser les pop-ups de LGLG V4 - Auteur ".
